# Defog — defog.cloud

Landing page for [Defog](https://defog.cloud), a free Chrome extension that filters noise from Norwegian news sites.

## Deploy

Static site — one `index.html` file. Host anywhere:

- **Cloudflare Pages**: Upload assets → connect `defog.cloud`
- **Vercel**: Import repo → done
- **GitHub Pages**: Settings → Pages → Deploy from branch

# Chrome Web Store — Defog

## Tittel (maks 45 tegn)

Defog — Filtrer nyheter på norske nettsider

## Kort beskrivelse (maks 132 tegn)

Skru av kategorier du ikke vil se på VG, NRK og Dagbladet. Én klikk, null støy.

## Full beskrivelse

Defog lar deg bestemme hva du ser på norske nyhetssider — uten å forlate siden.

Slå av kategoriene du ikke bryr deg om, og de forsvinner øyeblikkelig. Slå dem på igjen med ett klikk. Innstillingene huskes på tvers av faner og besøk.

**Støttede nettsider**
- VG og VGTV
- NRK
- Dagbladet
- DN

**Kategorier du kan filtrere**
- Sport
- Sladder / Rampelys
- Nyheter
- Økonomi
- Kultur / Underholdning
- Debatt / Meninger
- Video / TV
- AI-generert innhold

**Slik fungerer det**
Defog leser URL-ene og innholdet i artiklene på siden — ingen data sendes noe sted. Alt skjer lokalt i nettleseren din.

Ikonet i verktøylinjen viser en teller over hvor mange artikler som er skjult på siden du er på akkurat nå.
